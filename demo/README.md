# Demo Data

This folder contains sample transactional data for the `garment_production` app.

## File

- `garment_production_demo_data.json`: end-to-end example records from style creation to dispatch

## Suggested Use

- Use it as reference data while creating records manually in Desk.
- Use it as source material for a custom import or bench script.
- Map the placeholder values like item, supplier, customer, warehouse, and UOM to real masters in your site before import.

## Flow Covered

1. `Garment Style`
2. `Raw Fabric Receipt`
3. `Dyeing Batch`
4. `Cutting Plan`
5. `Contractor Job Work`
6. `Stitching Job Card`
7. `Quality Inspection`
8. `Production Dispatch`
9. `Wastage Entry`
